== Description ==

**Share Social Media** adds social media sharing icons to a post or page of your WordPress website and allow visitors to share your content on various social media sites. It supports multiple icons placement positions. Place social icons before or after page/post content. Also, you can place sticky icons on a right side or left side of your WordPress website.

### Share Social Media Features

* [Share Social Media Features](https://scriptstown.com/wordpress-plugins/share-social-media/)
* Social Icons to Page or Post
* Place Icons Before or After Content
* Sticky Social Share Icons
* Popular Social Media Icons
* Customize Social Sharing Icons
* Multiple Icons Placements
* Icons Design Settings

== Installation ==

**Share Social Media [Installation Guide]**

1. You can:
 * Upload the entire `share-social-media` folder to the `/wp-content/plugins/` directory via FTP.
 * Upload the zip file of plugin via *Plugins -> Add New -> Upload* in your WordPress Admin Panel.
 * Search **Share Social Media** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate the plugin through *Plugins* menu in WordPress Admin Panel.
3. Click on *Social Share* under *Settings* menu to configure the plugin.
4. Ready, now you can use it.

== Frequently Asked Questions ==

= How to configure icons placement? =
On your WordPress, go to *Settings* -> *Social Share* menu page to configure the placement of icons.
