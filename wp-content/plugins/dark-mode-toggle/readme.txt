== Description ==

**Dark Mode Toggle** is an excellent WordPress plugin to quickly add a dark / night mode toggle switch button on your WordPress site. The users can easily switch between a dark or light scheme by clicking on the toggle button. It is ready to use dark mode plugin that works out of the box and is compatible with all WordPress themes and offers many options to customize the position, width, height, and border radius of the toggle button. This plugin is very lightweight with minimal footprints. It is very fast and easy to use.

Dark mode can improve the accessibility of a website. A dark background with a light-colored text can reduce eye strain for many people and it's easier to read. This plugin allows users to switch between light and dark modes easily. Also, it saves the user's choice for the next page refresh.

This dark mode plugin is simple to use and works with all modern browsers and offers very high compatibility with all WordPress themes as well as page builder plugins. Also, it works well for the dark themes to toggle the light mode on the dark themes.

== Installation ==

**Dark Mode Toggle [Installation Guide]**

1. You can:
 * Upload the entire `dark-mode-toggle` folder to the `/wp-content/plugins/` directory via FTP.
 * Upload the zip file of plugin via *Plugins -> Add New -> Upload* in your WordPress Admin Panel.
 * Search **Dark Mode Toggle** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate the plugin through *Plugins* menu in WordPress Admin Panel.
3. Click on *Dark Mode* under *Appearance* menu to configure the plugin.
4. Ready, now you can use it.
